# 游戏公司2025-2026文化联动完整汇总

## 标签体系说明

### 游戏内载体
| 标签 | 含义 |
|------|------|
| `游戏内容植入` | 主线/支线剧情、关卡、地图区域融入文化元素 |
| `外观/时装` | 限定皮肤、时装、头饰等可穿戴虚拟内容 |
| `场景还原` | 游戏内高度还原实体建筑/文物/地理场景 |
| `原创音乐` | 游戏联动曲、非遗传承人演唱/演奏作品 |
| `纪录片/视频` | 制作文化纪录片、宣传片、MV |
| `实物复刻` | 将现实文物/非遗作品在游戏中数字化复刻 |
| `游戏活动` | 限时活动、奇遇、任务、打卡等互动玩法 |

### 游戏外延伸
| 标签 | 含义 |
|------|------|
| `传承人共创` | 与非遗/文化传承人直接合作创作内容 |
| `线下文旅` | 实景打卡、景区联动、线下沉浸体验 |
| `博物馆合作` | 与博物馆/文化机构官方联名合作 |
| `实物产出` | 制作联名实物周边（区别于游戏内虚拟品） |
| `文化科普` | 文化知识传播、教育向内容输出 |
| `品牌联名` | 与文化品牌/IP/节目跨界联名 |
